Project Explanation

Project

Our project is risk analysis website platform to help our community during these tough times of the pandemic. Many businesses are eager to open without knowing what risk they will bring and how successful they are. Our project aims to use google cloud risk analysis api, mongoDB, and machine learning to make our data as accurate as possible.


Plan Now Form

We created a form in html/php which would ask the user questions about the thier business. Some of the main criteria is location, employees, building size, and age group. All of these components would then be put into a machine learning algorithm coded in python.The algorithm would then calculate the success rate percentage of a business if it were to open during the pandemic. The lower the percentage, results in a lower business success which puts the business at a high risk level. Below you can find an explanation video of how the code works.

Machine Learning Algorithm



Google Cloud API

Using google maps API we were able to get information of local businesses in the community. By using our machine learning algorithm the output of a low or high risk would give us information on how one business can affect other businesses in the area. If there is a high risk then more local businesses in the area are also at risk. If there is a low risk then less local businesses are affected.
